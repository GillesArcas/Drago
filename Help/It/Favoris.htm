Drago - La lista dei preferiti

La lista dei file preferiti salva i nomi dei documenti che sono usati spesso o per i quali è necessario un riferimento. La lista dei preferiti può essere aperta dal menu File/Preferiti o con il pulsante sulla barra strumenti . Compare una finestra di dialogo, mostrando la lista di file e i seguenti pulsanti.

Pulsante Apri

Il pulsante Apri carica il documento selezionato. Questa opzione chiude la finestra di dialogo dei preferiti e salva ogni cambiamento apportato alla lista dei favoriti. Lo stesso risultato può essere ottenuto facendo doppio click su uno dei file nella lista.

Pulsante Aggiungi File

Il pulsante Aggiungi File è usato per aggiungere il file corrente ai preferiti.

Pulsante Aggiungi

Il pulsante Aggiungi è usato per selezionare un file da un disco e caricarlo nei preferiti.

Pulsanti e .

Questi due pulsanti sono usati per portare in alto o in basso il file selezionato all'interno della lista dei preferiti.

Pulsante Rimuovi

Il pulsante Rimuovi elimina il file selezionato dalla lista dei preferiti.

Pulsante Chiudi

Il pulsante Chiudi chiude la finestra di dialogo dei preferiti e salva qualsiasi cambiamento alla lista.

Pulsante Annulla

Il pulsante Annulla chiude la finestra di dialogo dei preferiti senza salvare i cambiamenti. I tasto Escape o un click sull'icona di chiusura della finestra di dialogo portano allo stesso risultato.

Pulsante Aiuto

Il pulsante Aiuto mostra questa pagina della guida.
