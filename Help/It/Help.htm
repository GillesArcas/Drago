Presentazione

Drago è un’applicazione gratuita per Windows dedicata al gioco del Go, con le seguenti caratteristiche:

  • Drago può esplorare e modificare i file in formato SGF.
  • Drago può gestire diversi file di gioco contemporaneamente (collezioni di partite o di problemi) o anche cartelle. E’ ad esempio possibile aprire un file sull’ultimo gioco selezionato, o per aprire un’anteprima, con testo e diagrammi, di tutti i giochi archiviati in un file o in una cartella.
  • Drago ha un’interfaccia a tabelle che permette di aprire diversi file contemporaneamente.
  • Drago può creare database di giochi usando la libreria libkombilo. Quindi, implementa la ricerca nelle informazioni di gioco, la ricerca di figure e quella delle firme.
  • Drago permette di giocare con un’AI gestita da un motore di gioco usando il protocollo GTP, inclusi GNU Go, Mogo, Go169, etc. [vanno scaricati a parte]
  • Drago può essere usato per rivedere le partite o per risolvere problemi, selezionandoli e giocandoli in vari modi.
  • Drago può essere usato per stampare delle partite, esportandole in formato RTF, PDF, HTML o Word, e può esportare i diagrammi in formato WMF, BMP, JPEG, GIF, PNG o ASCII.
  • Drago gestisce l’Unicode per l’interfaccia così come per i contenuti dei file del gioco.
