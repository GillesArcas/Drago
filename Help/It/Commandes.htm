Esci Chiude il programma

E' anche possibile aprire un file SGF:

  • trascinandolo nella finestra principale dal desktop di windows o dal browser,
  • facendo direttamente doppio click sul file se si è associato Drago quale applicazione di default per l'apertura dei file SGF nelle opzioni di windows.

Il menu Save, come il pulsante associato nella barra strumenti, non è disponibile quando il file corrente non è stato modificato.

Il comando Apri cartella permette di gestire le cartelle come file multipli di gioco. In altre parole, è possibile usare sulle cartelle tutti i comandi che possono essere usati sui file contenenti più partite, incluso:

  • ottenere un indice delle informazioni e delle tavole da gioco,
  • navigare tra una partita e l'altra,
  • stampare ed esportare,
  • selezionare ed archiviare in modalità "soluzione di problemi" e "partita".

Il menu Collezioni da accesso ai seguenti comandi:

  Aggiungi a... Aggiunge il file corrente ad un'altro
  Unisci... Seleziona e unisci una serie di file sgf
  Estrai corrente Estrae dal file la corrente partita
  Estrai tutto... Divide il file corrente in diversi file di gioco
  Crea l'albero di gioco... Crea l'albero di gioco dalle mosse e varianti di tutte le partite nella collezione

comandi da Tastiera

Drago è conforme all'ergonomia di Windows il che significa che è possibile usare l'applicazione utilizzando solo la tastiera. Qualsiasi comando può essere associato ad una scorciatoia dei tasti, dopo l'istallazione sono disponibili i seguenti comandi:

Seleziona una Partita

Ctrl-Freccia sinistra Partita precedente
Ctrl-Freccia destra Partita successiva
Ctrl-Home Prima partita
Ctrl-Fine Ultima partita

Navigazione nel gioco

Freccia sinistra Mossa precedente
Freccia destra Mossa successiva
Home Posizione di partenza
Fine Posizione finale
Freccia su Variante precedente
Freccia giù Variante successiva

Modifica

La posizione da modificare è identificata dal puntatore del mouse.

Alt-Freccia sinistra Colonna precedente
Alt-Freccia destra Colonna successiva
Alt-Freccia su Linea precedente
Alt-Freccia giù Linea successiva
Invio Gioca, o aggiunge una pietra o una annotazione nel punto indicato del puntatore del mouse
Ctrl-Invio Crea una nuova mossa in alternativa nella posizione del puntatore del mouse
Ctrl-z Annulla

Attivazione dei comandi da tastiera

I comandi da tastiera sono attivi solo se i nomi dei nodi e i commenti non sono in corso di modifica (il cursore lampeggiante non sia presente nel campo del nome del nodo o nella casella dei commenti).

Per selezionare il controllo attivo, dovete:

  • fare click sul controllo per il nome del nodo o per la casella di commento,
  • spostare il cursore sul goban per modificarlo,
  • o selezionare il controllo usando il tasto Tab.

Comandi supplementari

Alcuni comandi non sono disponibili da subito nell'interfaccia utente, ma possono essere aggiunti sia come scorciatoie da tastiera sia come pulsanti nella barra strumenti.

Nella versione attuale questi comandi sono:

  • L'Attiva/Disattiva coordinate, che permette di far comparire o scomparire dalla tavola da gioco le coordinate senza intervenire nel menu Opzioni,
  • l'Attiva/Disattiva numerazione mosse, con medesima funzione.

Questi due comandi possono essere trovati nella categoria Opzioni nelle schede Barra Strumenti e Scorciatoie della finestra di dialogo Opzioni.
