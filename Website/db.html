Drago - Database management

 

A very quick help to start using database features in Drago.


 

Commands

 

All commands are found in the database menu or the toolbar.

It is possible to open several databases at the same time.


Creating a database

 

Select files or folders

Searching by game information

  • result
  • date
  • or any SGF properties

Searching by pattern

  

Select pattern:

  • on board
  • or thumbnail

Start

Viewing results in game table

  

Columns can be configured.

Viewing results in thumbnails

  

After searching, patterns are displayed on boards.

Thumbnails can be configured.
 

Drago is installed with a very small database to start with (kisei.db, 119 games).
